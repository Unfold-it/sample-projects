# Unfold Sample Projects

Example applications and reference implementations built with the [Unfold](https://unfoldit.com) platform and MCP tools.

Each project is a standalone, runnable showcase of a real use case — designed to be cloned, explored, and adapted.

---

## Projects

### [`ai-academy/`](./ai-academy)

> **Train your engineers on AI. See who's learning.**

A complete AI readiness program for engineering teams — built on Unfold's assessment, goal, and analytics tools. Every engineer gets a personalized learning path. Every manager gets a real-time cohort dashboard.

**Covers:** skill assessment → targeted enrollment → cohort monitoring → AI-powered insights  
**Stack:** Next.js 15, TypeScript, Tailwind CSS  
**Runs locally:** `cd ai-academy && npm install && npm run dev`
